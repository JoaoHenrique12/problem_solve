DNS

Questões

  1. dnsdomainname Mostra o nome do domínio de seu sistema.
  2. localhost e 127.0.0.1 são os endereços de loopback do sistema que informam o endereço da próprima máquina, isto é útil dado que este endereço nunca muda, ele sempre está ligado e em 'alcance'.
  3. FQDN (em inglês, Fully Qualified Domain Name, ou em Português, Nome de Domínio Completamente Qualificado), é o nome de domínio absoluto. Sua interpretação não gera ambiguidade.
  4. Sim. Enquanto que o servidor DNS primário mantêm os registros o servidor secundário pode servir de cache.
  5. O DNS reverso retorna o nome dado o IP.
  6. A entrada MX é referente à servidores de e-mail. Para este tipo de entrada cada domínio só pode possuir uma. 
  7.         Uma resposta autoritativa é aquela dada por um servidor DNS que é considerado a fonte confiável para informações sobre um determinado domínio.
        É diferente de uma resposta não autoritativa, que é dada por um servidor que não é o responsável final pelo registro do domínio.
        Uma resposta autoritativa é geralmente considerada mais confiável e precisa.
  8. Um dns cache only mantêm o cache das requisições feitas, entretanto não sincroniza arquivos de dados.
