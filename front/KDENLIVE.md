# Kdenlive

## Keyboard short-cuts

```
Alt + right -> go until the end of clip.

```

## Audio Wave Form Filter

```
Used to show voice.

Must use master track.
```

## Transform

```
Used to zoom in and zoom out.

Rotate screen.
```

## Wipe

```

Used to uncover some image. It can be used to image appears from bottom,
up, be show from nothing.

```
