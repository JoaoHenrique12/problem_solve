# Data Structures

## Stack

```cpp
.size() .empty()
.push(el)
.top()
.pop()
```

## Queue
```cpp
.front()
.back()

.push(el)
.pop()
```

## Priority Queue

```cpp
priority_queue<int, vector<int>, less<int>> cont;

.empty()
.top()
.pop()
.push(el)
```

## Deque

```cpp
deque<int> d = {7, 5, 16, 8};
d.push_front(el)
d.push_back(el)

d.pop_front()
d.pop_back()
d.erase(iterator)
```
